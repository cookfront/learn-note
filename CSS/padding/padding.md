padding
========

`CSS`属性的`padding`属性设置一个元素四个方向所需要的内边距。`padding`区域是位于元素内容和`border`之间的空间。并且不允许负值哟。

 - 初始值：0
 - 应用于：所有的元素除了`display`属性值为`table-header-group`、`table-row-group`、`table-footer-group`、`table-row`、`table-column-group`、`table-column`和`table-inline`的元素
 - 继承：无
 - 百分比值：相对于包含块的宽度。

语法：

```c
padding: [ <length> | <percentage> ]{1,4}
```

`padding`也可以一次设置1到4个值，和`margin`一样：[margin](https://github.com/cookfront/learn-note/blob/master/CSS/margin/margin.md)
