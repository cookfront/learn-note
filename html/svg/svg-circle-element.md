`SVG`的`<circle>`元素用于绘制圆形。看看例子：

```c
<svg xmlns="http://www.w3.org/2000/svg"
    xmlns:xlink="http://www.w3.org/1999/xlink">

    <circle cx="40" cy="40" r="24" style="stroke:#006600; fill:#00cc00"/>

</svg>
```

注意这里的`cx`，`cy`表示的是圆的圆心的位置，而`r`属性是圆的半径。
